# Student Grade Analyzer – Data Analysis with Python

## Description
A comprehensive data analysis project that analyzes student grade data using Python. This project demonstrates data manipulation, statistical analysis, and data visualization techniques.

## Live Demo
Open in Google Colab: [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/maxtinka/student-grade-analyzer/blob/main/analysis_notebook.ipynb)

## Features
- Load and explore student grade data (15 students, 4 subjects)
- Calculate statistics (mean, median, standard deviation, min, max)
- Grade distribution analysis with letter grades
- Attendance vs grades correlation analysis
- Data visualizations:
  - Bar chart of student averages
  - Line chart of subject averages
  - Pie chart of grade distribution
  - Scatter plot with trend line (attendance vs grades)
  - Box plot of grade distribution by subject
- Export results to CSV and text files
- Save all visualizations as PNG images

## Technologies Used
- **Python 3** – Programming language
- **pandas** – Data manipulation and analysis
- **numpy** – Numerical operations
- **matplotlib** – Data visualization
- **seaborn** – Enhanced visual styling
- **Jupyter Notebook** – Interactive environment
